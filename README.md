## Sobre mim

Comecei a programar em 2020 desenvolvendo jogos com **C#**, o que me deu uma base sólida em lógica e estrutura de código. Hoje, como estudante de **Ciência da Informação na UFSC**, direciono esse background técnico para as áreas de **análise de dados, automação e integração de sistemas**.

Tenho experiência prática com Python para automação de processos, pipelines de ETL com dados públicos e construção de dashboards com Power BI. Busco oportunidades onde possa contribuir com projetos de dados e continuar crescendo profissionalmente.
